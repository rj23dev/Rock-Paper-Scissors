# Rock Paper Scissors

This is a simple **Rock Paper Scissors** game built with JavaScript.  
The game lets the user play against the computer until one reaches 5 points, and declares the final winner based on the score.

## Features

- Plays Rock, Paper, Scissors untill one player reaches 5 points
- Declares a final winner at the end

## How it works

1. The computer randomly chooses between `rock`, `paper`, or `scissors`.
2. The user clicks a button to choose their entry for each round.
3. The game compares the two choices and determines a winner for each round.
4. The game continues until either the player or computer reaches 5 points, then the final scores are shown.



## How to run locally

1. Clone the repository or download the source files.
2. Open the `index.html` file in your web browser.
3. The game will start immediately.
